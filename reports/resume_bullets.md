# Resume Bullets

- Built an end-to-end customer churn prediction system using Python, pandas, scikit-learn, and Streamlit, including data cleaning, leakage prevention, model training, threshold tuning, batch scoring, local explanations, and interactive deployment.
- Improved churn detection with a cost-sensitive Logistic Regression pipeline, reaching 92.5% recall and 0.841 ROC AUC on a stratified test set.
- Designed an ROI-based retention threshold that converts churn probabilities into contact recommendations, estimates campaign value, and flags feature drift for uploaded customer files.
